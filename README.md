### 👨🏽‍💻 Sobre Mim

Sou **Engenheiro de Software (Bacharel)** pela **PUC Minas**, com forte atuação em **Java (Spring Boot)**, **Python** e **React**. 

Meu foco vai além do código: busco arquiteturas resilientes, escaláveis e a resolução de problemas de negócio através da tecnologia.

* 🧠 **Interesses:** Domain-Driven Design (DDD), Event-Driven Architecture, DevOps e Performance Tuning.
* 💡 **Superpoderes:** Engenharia reversa para troubleshooting (ex: *race conditions*), mentoria técnica e padronização de arquitetura.
* 🚀 **Projetos Recentes:** Sistema de gestão para a ONG FUTEARTE BH (Full Stack + Azure) e Mobilidade Carnaval BH.
